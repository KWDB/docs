# 数据库版本查看

连接数据库后，用户可以使用 `SELECT version()`  语句查看 KWDB 的版本信息。

```sql
SELECT version();
```

执行成功后，控制台输出以下信息：

```sql
                                            version
------------------------------------------------------------------------------------------------
  KaiwuDB 2.0.3.2 (x86_64-linux-gnu, built 2024/04/15 03:12:30, go1.15.14, gcc 9.4.0)
(1 row)
```
