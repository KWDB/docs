# KWDB 3.2.1 发版说明

KWDB 是一款面向 AIoT 场景的分布式、多模融合的数据库产品，支持在同一实例同时创建时序库和关系库，并融合处理多模数据，具备千万级设备接入、百万级数据秒级写入、亿级数据秒级读取等时序数据高效处理能力，具有稳定安全、高可用、易运维等特点。面向工业物联网、数字能源、车联网、智慧产业等领域，提供一站式数据存储、管理与分析的基座。

KWDB 3.2.1 版本在保持原有特性的基础上，在数据类型和时区设置等方面引入了新特性与优化。

## 新增特性

## 数据管理与查询

### 数据类型扩展

- **CITEXT 类型支持**：关系数据新增大小写不敏感的文本类型 `CITEXT`。查询时不区分大小写，支持大小写不敏感的模式匹配。

### 全局时区设置

- 支持运行期通过集群参数动态配置全局时区：客户端连接未携带时区时，`TIMESTAMPTZ`数据采用集群配置时区；连接显式指定时区时，以连接携带的时区作为会话时区。

## 重要变更

### 开发工具

- **KaiwuDB 开发者中心**：新增 `CITEXT` 类型支持

## 升级说明

- **多副本集群**：支持 KWDB 3.0.0、3.1.0 和 3.2.0 版本离线升级至 3.2.1

- **单副本集群**：支持 KWDB 3.0.0、3.1.0 和 3.2.0 版本离线升级至 3.2.1

- **单机版本**：支持 KWDB 3.0.0、3.1.0 和 3.2.0 版本离线升级至 3.2.1

- **KWDB 2.x 版本**：支持通过导入导出方式升级至 3.2.1
